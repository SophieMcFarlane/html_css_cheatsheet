HTML and CSS Cheatsheet

HTML Table Reference

Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML tables
<thead> Table Head The set of rows defining the column headers in a table
<tbody> Table The set of rows containing actual data
<tr> Table Row The table row container
<td> Table Data The table row container for the data
<tfoot> Table Foot The set of rows defining the footer in a table

Table Attributes

Attribute Name Description
colspan Column Span Defines how many columns a td element should span
rowspan Row span Defines how many rows a td element should span

HTML & CSS List Reference

HTML List Tags

Tag Name Description
<ul> Unordered List Used to form an unordered list
<ol> Ordered List Used to form an ordered list
<li> List item Used to add items to a list
<dl> Description List Used to form an description list
<dt> Description term Used to add a term to a description list
<dd> Description term Used to describe each term in a description list

HTML List types

Type Name Description
type="1" Numbers List items will be numbered with numbers(default)
type="A" Uppercase List items will be numbered with uppercase letters
type="a" Lowercase List items will be numbered with lowercase letters
type="I" Uppercase Roman List items will be numbered with uppercase roman numbers
type="i" Lowercase Roman List items will be numbered with lowercase roman numbers

CSS List Attributes

Attribute Name Description
list-style-type: disc Disc Sets the list item marker to a bullet (default)
list-style-type: circle Circle Sets the list item arker to a circle
list-style-type: square Square Sets the list item marer to a square
list-style-type: none None The list items will not be marked

HTML <script> Tag Reference

Script Attributes

Attribute Name Description
defer Defer scripts will be executed after the HTML file is completely parsed
async Async HTML parser will continue parsing the rest of the HTML as the script is downloaded in the background
